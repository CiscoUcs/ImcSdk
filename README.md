# Python SDK for Cisco IMC

* Apache License, Version 2.0 (the "License")

## Installation
```
    - Install pip (skip if pip is already available):
        wget https://bootstrap.pypa.io/get-pip.py
        python get-pip.py

    git clone https://github.com/ciscoucs/imcsdk.git
    cd imcsdk
    make install
```
